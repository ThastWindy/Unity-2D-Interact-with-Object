# Unity-2D-Interact-with-Object
Unity 2D Interact / Use Object

## How to use
1. Add `CurrentObject.cs` to your player's interact collider (Make sure to include Collider2D)
2. Add `Object.cs` to the object you wish to interact with (Make sure to include Collider2D as well)
3. Assign value to your likings
4. Done!
